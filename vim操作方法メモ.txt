vim操作方法メモ

h,j,k,l
　→ カーソルを1つずつ移動する。

x
　→ カーソル位置の文字を削除する。

i
　→ カーソル位置で入力モードに入る。

o
　→ カーソル行の下に行を挿入し、入力モードに入る。

O
　→ カーソル行の上に行を挿入し、入力モードに入る。

a
　→ カーソルの1つ後ろに移動し、入力モードに入る。

A
　→ カーソル行の末尾に移動して入力モードに入る。

d
　→ 文字を削除する。モーションは以下。

    dw
　  　→ カーソル位置にある単語を削除する。(空白を含む)

    de
　　　→ カーソル位置にある単語を削除する。(空白を含まない)

    d$
　　　→ カーソル位置から行の末尾までを削除する。

    dd
    　→ カーソル行を削除する。

^ , 0(ゼロ)
　→ 文頭に移動する。

u
　→ 直前のコマンドを取り消す。

U
　→ 直前のコマンドで変更した行をもとに戻す。

ctrl + r
　→ 直前の取り消したコマンドをもとに戻す。

y
　→ 選択したテキストをコピーする。

    yw
    　→ カーソル位置のある単語をコピーする。

    y$
    　→ カーソル位置から行末までをコピーする。

    yy
    　→ カーソル行をコピーする。

p
　→ 最後にコピーや削除された行をカーソル位置に貼り付ける。

r
　→ カーソル位置の文字を置換する。r を入力したあとに置換後の文字を入力する。

R
　→ カーソル位置から文字を置換していく。入力終了はESC。

c
　→ カーソル位置の文字列を削除し、入力モードに入る。モーションは以下。

    cw
    　→ カーソル位置から単語の末尾まで削除し、入力モードに入る

    c$
    　→ カーソル位置から行末まで削除し、入力モードに入る。

ctrl + g
　→ ファイル名や行番号、ページのどのあたりにいるかを表示する。

G
　→ ファイルの末尾に移動する。

gg
　→ ファイルの先頭に移動する。

[行番号]G
　→ ファイルの指定した行番号の位置に移動する。

ctrl + o
　→ 直前のカーソル位置に戻る。

/[文字列]
　→ 指定した文字列を下へ検索していく。n で次(下)の文字列、N で前(上)の文字列の位置へ移動する。

?[文字列]
　→ 指定した文字列を上へ検索していく。n で次(上)の文字列、N で前(下)の文字列の位置へ移動する。

%
　→ カーソル位置の括弧に対応している括弧の位置へ飛ぶ。

:s/old/new
　→ oldをnewに置換する。カーソル行の最初に該当した文字列だけを置換。他に以下のような指定がある。

    :s/old/new/g
　  　→ カーソル行の該当する文字列全てを置換。

    :#,#s/old/new/g
    　→ #から終了行までのoldを全てnewに置換する。# には行番号が入る。

    :#,#s/old/new/gc
    　→ #から終了行までのoldを、1つ1つ置換するかの確認を取りながら全てnewに置換する。# には行番号が入る。

    :%s/old/new/g
    　→ ファイル全体のoldを全てnewに置換する。

    :%s/old/new/gc
    　→ ファイル全体のoldを、1つ1つ置換するかの確認を取りながら全てnewに置換する。

:!
　→ シェルコマンドを実行する。例えば、:!ls と入力すれば、vimを使いながらでも直下のフォルダを確認することができる。

:w [ファイル名]
　→ 現在のファイルの状態を指定したファイルに保存する。

v
　→ ビジュアルモードに入る。選択した範囲に様々な操作を適用できる。例えば、d で選択した範囲を削除したり、
　　 :w [ファイル名] で選択した範囲だけをファイルに保存することができる。

:r [ファイル名]
　→ カーソル位置にファイルの中身を挿入する。

:set オプション
　→ 様々なオプションを設定する。解除するときは :set noxxx とする。

    :set ic(ignorecase)
    　→ 検索の際に大文字小文字を区別しない。

    :set hls(hilightsearch)
    　→ 検索に一致した文字列をハイライト表示する。

    :set is(incsearch)
    　→ 検索フレーズに部分マッチしている部分を表示する。

:help
　→ ヘルプを表示する。

b
　→ 前の単語へジャンプしていく。

ctrl + d
　→ コマンドの一覧を表示する。例えば、:e と打った状態でctrl + d を押すと、e から始まるコマンドの一覧が表示される。

tab
　→ コマンドやファイル名を補完する。

その他メモ

・多くのコマンドは「オペレータ」と「モーション」からテキストに変更を加える。例えば d コマンドの場合は以下のようになる。

    d モーション(w,e,$など)

　モーションキーを押すと、対象となる場所へカーソルが移動する。wなら次の単語の先頭、eなら次の単語の末尾など

・オペレータとカウントをタイプすると、その操作が複数回繰り返される。
　例えば、d3w　とすると、dw の操作を3回行うことになる。書式は以下の通り。

    オペレータ　[数値]　モーション

・vimの多くの機能は初期状態で使用不可となっている。これを有効にするには「vimrcファイル」を作成し、編集する必要がある。
　:edit ~/.vimrc コマンドで vimrcファイルを編集することができる。
